# APB-Protocol-Verilog
Verilog implementation and verification of AMBA APB protocol with read/write transaction support and testbench validation.

## Overview
This project implements and verifies a simple APB (Advanced Peripheral Bus) module using Verilog HDL.

## Features
- APB Write Transfer
- APB Read Transfer
- PSEL, PENABLE, PWRITE handling
- PREADY support
- Functional verification using testbench

- ## Files
- apb_protocol.v : DUT
- apb_protocol_tb.v : Testbench

- ## Tools Used
- Verilog HDL
- ModelSim / QuestaSim
- Quartus prime
